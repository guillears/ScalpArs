LONG signals require RSI at or below threshold (oversold). SHORT signals require RSI at or above threshold (overbought). EMA stacking is always required.

EMA Stacked for LONG: EMA5 > EMA8 > EMA13 > EMA20 (bullish). EMA Stacked for SHORT: EMA5 < EMA8 < EMA13 < EMA20 (bearish).

Price vs EMA20 filter

Only enter LONGs when price > EMA20; only enter SHORTs when price < EMA20

EMA20 Slope filter (6-candle / 30min)

Only enter LONGs when EMA20 is rising vs 6 candles ago; only enter SHORTs when falling

Macro Trend Regime (EMA20)

EMA20 rising = BULLISH (LONGs only), falling = BEARISH (SHORTs only), flat = NEUTRAL (configurable)

BTC Global: uses BTC's EMA20 regime to gate ALL pairs (requires BTC + pair regime alignment)

RSI & ADX Filter

LONG RSI ≥ 0 or ≤ 100 to disable. SHORT RSI ≤ 100 or ≥ 0 to disable.

RSI Momentum Filter

Block LONG if RSI is falling (vs 3 candles ago). Block SHORT if RSI is rising. Prevents entries against RSI momentum direction.

EMA Gap Expanding Filter

Block entry if EMA5-EMA8 gap is compressing (current gap ≤ previous candle). Only enter when momentum is expanding.

Momentum signals are evaluated BEFORE EMA-stack signals. LONG: EMA5 > EMA8, gap = ((EMA5-EMA8)/EMA8)*100. SHORT: EMA5 < EMA8, gap = ((EMA8-EMA5)/EMA5)*100.
